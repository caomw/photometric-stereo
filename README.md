Photometric Stereo
==================

## Dependencies

- Python 2.7
- OpenCV 2.4 / 3.x
- Numpy (`pip install numpy`)
- Cython (`pip install Cython`)
- PyOpenGL (`pip install PyOpenGL PyOpenGL-accelerate`)
- MVE (Multi-View Environment)

## Build

    cd mve
    python setup.py build_ext --inplace

## Run

    python warp.py -iIMANE_NAME SCENE PLY_MESH
